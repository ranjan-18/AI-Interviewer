# AI Interview Buddy - Frontend

This is the immersive, cinematic React frontend for the AI Interview Buddy application.

## 🚀 Key Features

*   **Cinematic HUD**: Optimized for large screens (1850px max-width) with an 80/20 layout focusing on technical transcript.
*   **Voice Interactivity**: Real-time STT/TTS integration with visualizer feedback.
*   **Framer Motion Animations**: Smooth transitions and high-fidelity message bubble entrances.
*   **Glassmorphic UI**: Futurisitc dark mode aesthetic.

## 🛠️ Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

The frontend expects the backend to be running at `http://localhost:8000`.
